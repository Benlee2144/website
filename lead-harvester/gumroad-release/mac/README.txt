QUICK START
-----------
1. Double-click "LeadHarvester.dmg" to open it
2. Drag "LeadHarvester" to the Applications folder
3. Open Applications and double-click LeadHarvester
4. That's it!

FIRST TIME SETUP
----------------
When you first run LeadHarvester, macOS may show a security warning:
"LeadHarvester can't be opened because it is from an unidentified developer"

To proceed:
1. Open System Preferences > Security & Privacy
2. Click "Open Anyway" next to the LeadHarvester message
3. Or: Right-click the app > Open > Click "Open" in the dialog

This is normal for apps not from the Mac App Store.

HOW TO USE
----------
1. Create a New Project
   - Click "New Project" and give it a name (e.g., "NYC Restaurants")

2. Start Scraping
   - Enter a search query like "restaurants in New York"
   - Click "Start Scraping"
   - The app will extract business leads from Google Maps

3. View Your Leads
   - All collected leads appear in your project
   - Each lead includes: Business name, address, phone, website, ratings, etc.

4. Export Your Data
   - Click "Export" to download your leads
   - Choose CSV or JSON format
   - Open CSV files in Excel, Numbers, Google Sheets, or any spreadsheet app

TROUBLESHOOTING
---------------
App won't start?
- Make sure you've moved the app to Applications folder
- Try the right-click > Open method
- Check System Preferences > Security & Privacy

"App is damaged" error?
- Open Terminal and run:
  xattr -cr /Applications/LeadHarvester.app
- Then try opening the app again

Scraping not working?
- Check your internet connection
- Try a different search query
- Some searches may take longer than others

Data not saving?
- The app saves data to: ~/Library/Application Support/LeadHarvester
- Make sure you have write permissions to this folder
