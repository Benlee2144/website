QUICK START
-----------
1. Extract this ZIP to any folder (e.g., Desktop or Documents)
2. Open the extracted folder
3. Double-click "LeadHarvester.exe" to run the app
4. That's it! No installation required.

FIRST TIME SETUP
----------------
When you first run LeadHarvester, Windows may show a security warning:
"Windows protected your PC"

To proceed:
1. Click "More info"
2. Click "Run anyway"

This is normal for apps not from the Microsoft Store.

HOW TO USE
----------
1. Create a New Project
   - Click "New Project" and give it a name (e.g., "NYC Restaurants")

2. Start Scraping
   - Enter a search query like "restaurants in New York"
   - Click "Start Scraping"
   - The app will extract business leads from Google Maps

3. View Your Leads
   - All collected leads appear in your project
   - Each lead includes: Business name, address, phone, website, ratings, etc.

4. Export Your Data
   - Click "Export" to download your leads
   - Choose CSV or JSON format
   - Open CSV files in Excel, Google Sheets, or any spreadsheet app

-------------------

TROUBLESHOOTING
---------------
App won't start?
- Make sure you extracted the ZIP file (don't run from inside the ZIP)
- Try running as Administrator (right-click > Run as administrator)

Scraping not working?
- Check your internet connection
- Try a different search query
- Some searches may take longer than others

Data not saving?
- The app saves data to: %APPDATA%\LeadHarvester
- Make sure you have write permissions to this folder
